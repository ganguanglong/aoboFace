Face++ 人脸 SDK（Android）
===

1. `facepp_Android/` 目录下存放的是 Face++ 人脸 SDK 的 C 接口 SDK；
2. `licmgr_Android/` 目录下存放的是授权管理的 C 接口 SDK；(非联网授权 SDK 则不需要)
3. 如果您需要了解该 SDK 的接口及相关信息请移步至 `doc/` 目录下；
4. 如果您想要快速集成，或者学习使用该 SDK 做一个简单的 Demo，请移步至 https://github.com/FacePlusPlus/MegviiFacepp-Android-SDK。
